SUVBC player of the week plugin

Installation:
1) Once you have successfully downloaded SUVBC_player_of_the_week.zip, place it somewhere easy to find, such as your desktop.

2) On your WordPress dashboard, navigate to Plugins > Add New > Upload. (If you are having trouble finding Upload, it is located underneath the Install Plugins title.)

3) Now click Browse... or Choose File and find the SUVBC_player_of_the_week.zip file. Once you have selected SUVBC_player_of_the_week.zip, click Open, then Install Now. The following read-out should appear to ensure that successful installation occurred: 
“Unpacking the package… 
Installing the plugin… 
Plugin installed successfully.”

4) Click Activate Plugin. After doing so, you will see “Plugin activated” in a box at the top of your screen. This plugin has now been successfully installed and is ready to be used. Continue to the set-up below to lean how to properly add and detelete players.

	NOTE: If you are using localhost, just extract the SUVBC_player_of_the_week.zip into your	www/wordpress/wp-content/plugins and then activate it in your admin plugins page.

-----------------------------------------------------------------------------------------------------------------
SET-UP
1) On the dashboard you will find a tab that says Weekly player. Click this then add new

2) The title of this post does not matter. Enter [suvbc_mcm] as the content and then publish.

-----------------------------------------------------------------------------------------------------------------
ADD PLAYERS
1) On the bottom of the dashboard, click SUVBC Weekly Player

2) Here you will be able to add and remove players from this table. Click Browse to find the right image to use

3) A wordpress media uploader will appear. Select the right image or upload a new file

4) Once you have selected this image, click save image. The image source will be filled out.

5) Enter in the players name and some information about them and then click save.

6) If everything was done correctly, a player add message will show and you will see the name shown under "Added Players"

-----------------------------------------------------------------------------------------------------------------
DELETE PLAYERS
1) Select the player you wish to remove from the table by clicking on their name. It will become highlighted  

2) Click Delete

3) Press ok if you want to delete this player. You will now see the player is removed from the Added Players table
